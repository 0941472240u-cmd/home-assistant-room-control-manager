# Changelog

## 0.1.0 - 2026-06-22

### Added

- UI Config Flow และ Reconfigure Flow สำหรับหลายห้อง
- Options Flow แบบ Reload หนึ่งครั้งด้วย `OptionsFlowWithReload`
- Entity Selectors และ Validation
- Ownership checking พร้อม Lock สำหรับช่วงตรวจสอบสุดท้าย
- Sensor sharing warning และ Actuator/IR ownership blocking
- Availability monitoring แยกตามโมดูล
- Repair Issues สำหรับ Entity ที่หาย unavailable หรือการตั้งค่าที่ไม่ครบ
- Dry-run decision evaluator ที่ไม่มี Service Call
- Startup hold, delays, thresholds และ sensor aggregation
- Diagnostics พร้อมปกปิดชื่อห้อง Area และ Entity ID
- Status sensors และ binary sensors
- ภาษาอังกฤษและภาษาไทย รวม `strings.json`
- HACS, Hassfest, test และ release workflows
- คู่มืออัปโหลด GitHub ติดตั้ง HACS และทดสอบ Dry Run
